Flappy Pipes v1
by Andrew "Yangmaster" Yang
Released: 23-March-2014

REQUIREMENTS
--------------------------------------------------------------------------------
- Python 2.7.5 and PyGame 1.9.1
  Download Python at: http://www.python.org/download/releases/2.7/
  Download PyGame at: http://www.pygame.org/download.shtml
--------------------------------------------------------------------------------

INSTRUCTIONS
--------------------------------------------------------------------------------
- Run skeleton.py to begin the game.
- Plays just like the popular game "Flappy Bird."
--------------------------------------------------------------------------------

FUTURE ADDITIONS
--------------------------------------------------------------------------------

- Increasing difficulty as time goes on
